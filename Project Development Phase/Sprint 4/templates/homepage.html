The project aims to develop a system that converts the sign language into a human hearing voice in the desired language to convey a message to normal people. We are making use of a convolution neural network to create a model that is trained on different hand gestures. This web app is built using this model. This app enables deaf and dumb people to convey their information using signs which get converted to human-understandable language and speech is given as output.
